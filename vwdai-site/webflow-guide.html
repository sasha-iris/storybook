1 Первые шаги в Webflow
Сделай это до начала работы — иначе шрифты и анимации не появятся
Шаг 1 — Подключи шрифты Google
1
В Webflow
Открой Project Settings
В левом верхнем углу нажми на логотип Webflow → Project Settings → вкладка Custom Code
2
Поле «Head Code»
Вставь ссылки на шрифты и иконки
Скопируй целиком и вставь в поле «Head Code»:
<!-- Шрифты --> <link href="https://fonts.googleapis.com/css2?family=Lexend:wght@200;300;400;500;600;700;800&family=DM+Sans:wght@300;400;500;600&display=swap" rel="stylesheet"> <!-- Иконки Phosphor --> <script src="https://unpkg.com/@phosphor-icons/web@2.1.1"></script>
3
Сохрани
Нажми Save Changes
После этого в поле выбора шрифта появятся Lexend и DM Sans.
Шаг 2 — Добавь CSS переменные (цвета)
В том же поле «Head Code» добавь после предыдущего кода:
<style> :root { --violet: #6951F4; --violet-light: #9b8fff; --purple: #C35DFD; --cyan: #4DD4FF; --ink: #0c0e1a; --warm: #131628; } </style>
💡
После этого в Webflow ты сможешь вводить var(--violet) прямо в поля цвета — и это будет работать.

10 Анимации
Большинство анимаций — готовый CSS, вставь один раз в Head Code и они заработают
ℹ️
Вставь весь CSS анимаций в Project Settings → Custom Code → Head Code — в блок <style> который ты добавлял в Главе 1. Вставь перед закрывающим тегом </style>.
Анимация 1 — Появление Hero при загрузке (fadeUp)
Что происходит:
Текст в Hero снизу вверх появляется с задержкой — сначала badge, потом заголовок, потом подзаголовок, потом кнопки.
Задержки для каждого элемента Hero:
Badge (eyebrow)задержка 0.1 сек
Заголовок H1задержка 0.2 сек
Подзаголовокзадержка 0.3 сек
Кнопкизадержка 0.4 сек
Trust barзадержка 0.5 сек
CSS — добавь в Head Code:
@keyframes fadeUp { from { opacity: 0; transform: translateY(24px); } to { opacity: 1; transform: translateY(0); } } .hero-eyebrow { animation: fadeUp 0.7s ease 0.1s both; } .hero-headline { animation: fadeUp 0.7s ease 0.2s both; } .hero-sub { animation: fadeUp 0.7s ease 0.3s both; } .hero-actions { animation: fadeUp 0.7s ease 0.4s both; } .hero-trust { animation: fadeUp 0.7s ease 0.5s both; }
⚠️
Каждому элементу Hero в Webflow дай соответствующий класс: hero-eyebrow, hero-headline и т.д.
Анимация 2 — Плавающие теги (tagFloat)
Что происходит:
5 тегов в Hero-секции (Brand Safe ✓, Verified, LIVE, AI Compliant, Trend ↑) медленно поднимаются на 8px и опускаются. Каждый с разной скоростью — выглядит органично.
Движениевверх на 8px и обратно
Тег 1 (Brand Safe)6 сек · задержка 0
Тег 2 (Verified)7 сек · задержка 1 сек
Тег 3 (LIVE)8 сек · задержка 0.5 сек
Тег 4 (AI Compliant)6.5 сек · задержка 1.5 сек
Тег 5 (Trend ↑)9 сек · задержка 0.8 сек
@keyframes tagFloat { 0%, 100% { transform: translateX(-50%) translateY(0px); } 50% { transform: translateX(-50%) translateY(-8px); } } .viz-tag-1 { animation: tagFloat 6s ease-in-out 0s infinite; } .viz-tag-2 { animation: tagFloat 7s ease-in-out 1s infinite; } .viz-tag-3 { animation: tagFloat 8s ease-in-out 0.5s infinite; } .viz-tag-4 { animation: tagFloat 6.5s ease-in-out 1.5s infinite; } .viz-tag-5 { animation: tagFloat 9s ease-in-out 0.8s infinite; }
⚠️
Каждому тегу в Webflow дай класс viz-tag-1 ... viz-tag-5. Теги должны быть Position: Absolute внутри контейнера с Position: Relative.
Анимация 3 — Рост progress bars (barGrow)
Что происходит:
Цветные полоски в Score Panel вырастают от 0% до нужной ширины при загрузке страницы.
Brand Alignment88% · зелёный
FTC Compliance62% · красный
Hate / Violence75% · красный
Political Content48% · оранжевый
@keyframes barGrow { from { width: 0; } to { width: var(--w); } } /* Каждой полоске задай класс и переменную --w: */ .bar-1 { --w: 88%; animation: barGrow 2s ease forwards; } .bar-2 { --w: 62%; animation: barGrow 2s ease forwards; } .bar-3 { --w: 75%; animation: barGrow 2s ease forwards; } .bar-4 { --w: 48%; animation: barGrow 2s ease forwards; }

⚠ Важно
Орбиты — это SVG элемент. Его нельзя нарисовать в Webflow визуально. Вставь через HTML Embed целиком из исходного кода сайта.

@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .orbit-ring-1 { animation: spin 22s linear infinite; transform-origin: 240px 240px; } .orbit-ring-2 { animation: spin 34s linear infinite reverse; transform-origin: 240px 240px; } .orbit-ring-3 { animation: spin 50s linear infinite; transform-origin: 240px 240px; }

/* ── VwD All Animations ── */ @keyframes fadeUp { from { opacity: 0; transform: translateY(24px); } to { opacity: 1; transform: translateY(0); } } @keyframes tagFloat { 0%, 100% { transform: translateX(-50%) translateY(0px); } 50% { transform: translateX(-50%) translateY(-8px); } } @keyframes barGrow { from { width: 0; } to { width: var(--w); } } @keyframes liveBlink { 0%, 100% { opacity: 1; } 50% { opacity: 0.35; } } @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } @keyframes breathe { 0%, 100% { transform: scale(1); opacity: 0.7; } 50% { transform: scale(1.08); opacity: 1; } } /* Hero fadeUp delays */ .hero-eyebrow { animation: fadeUp 0.7s ease 0.1s both; } .hero-headline { animation: fadeUp 0.7s ease 0.2s both; } .hero-sub { animation: fadeUp 0.7s ease 0.3s both; } .hero-actions { animation: fadeUp 0.7s ease 0.4s both; } .hero-trust { animation: fadeUp 0.7s ease 0.5s both; } /* Floating tags */ .viz-tag-1 { animation: tagFloat 6s ease-in-out 0s infinite; } .viz-tag-2 { animation: tagFloat 7s ease-in-out 1s infinite; } .viz-tag-3 { animation: tagFloat 8s ease-in-out 0.5s infinite; } .viz-tag-4 { animation: tagFloat 6.5s ease-in-out 1.5s infinite; } .viz-tag-5 { animation: tagFloat 9s ease-in-out 0.8s infinite; } /* Orbit rings */ .orbit-ring-1 { animation: spin 22s linear infinite; transform-origin: 240px 240px; } .orbit-ring-2 { animation: spin 34s linear infinite reverse; transform-origin: 240px 240px; } .orbit-ring-3 { animation: spin 50s linear infinite; transform-origin: 240px 240px; } /* Progress bars */ .bar-1 { --w: 88%; animation: barGrow 2s ease forwards; } .bar-2 { --w: 62%; animation: barGrow 2s ease forwards; } .bar-3 { --w: 75%; animation: barGrow 2s ease forwards; } .bar-4 { --w: 48%; animation: barGrow 2s ease forwards; } /* Live dot */ .live-dot { animation: liveBlink 1.4s ease-in-out infinite; } /* Hero glow breathe */ .hero-glow { animation: breathe 5s ease-in-out infinite; }
