Webflow Setup — пошагово

1

Подключи Google Fonts

Project Settings → Fonts → Add Google Font → выбери Lexend (200–800) и DM Sans (300–600). Лексенд — для всех заголовков H1–H4. DM Sans — для body и UI.

2

Добавь Phosphor Icons CDN

Project Settings → Custom Code → Head Code — вставь:

<script src="https://unpkg.com/@phosphor-icons/web@2.1.1"></script>

Иконки вставляются через HTML Embed: <i class="ph-light ph-eye"></i>

3

Вставь CSS-переменные и @keyframes

Project Settings → Custom Code → Head Code — вставь после script тега:

<style> :root { --violet: #6951F4; --violet-light: #9b8fff; --purple: #C35DFD; --cyan: #4DD4FF; --ink: #0c0e1a; --warm: #131628; --light-bg: #f5f6fa; --light-surface: #ffffff; --light-mid: #52526a; --light-secondary: #8888a0; --light-border: rgba(10,10,15,0.08); } </style>
4

Секции с gap: 1px (Pillars, Steps, Testimonials)

Создай обёртку-div с фоновым цветом rgba(10,10,15,0.08), display Grid, gap 1px. Карточки внутри — белый фон. Эффект: тонкие разделители между карточками без явного border.

5

Большие фоновые числа в Step карточках

Внутри каждой карточки: добавь Text Block с текстом "01", "02" и т.д. → position: absolute, top: 16px, right: 20px, font-size: 96px, Lexend 800, color: rgba(105,81,244,0.07). Родительская карточка — position: relative. Всё через Webflow Designer, кода не нужно.

6

Radar SVG и Score Panel

Обе части Hero-визуализации — сложные SVG с JS-анимацией. Используй HTML Embed block и вставь код как есть из index.html. Расположи через absolute positioning внутри relative родителя.

7

Плавающие теги (FTC, Followers и т.д.)

Каждый тег — отдельный Div, position: absolute, с кастомным CSS-классом. Анимация tagFloat добавляется через Custom CSS. Координаты смотри в таблице ниже.

Тегtopleft
FTC: 3 Violations8%52%
744.8K Followers26%90%
⚠ Hate Content: High62%16%
Brand Safe ✓44%80%
Political: Moderate26%14%

Все теги имеют transform: translateX(-50%) для центровки по left-оси.

8

barGrow animation для прогресс-баров

В Webflow используй Interaction: On Page Load → target .vsp-fill → width от 0% до нужного значения, 2s ease. Или через Custom CSS c animation: barGrow 2s ease forwards и CSS-переменной --target-width.

① Герой — появление текста (fadeUp)
Что делает:
Каждый элемент героя появляется снизу вверх с задержкой. Eyebrow → H1 → Subtitle → Кнопки → Trust bar
В Webflow:
Interactions → Page Load → Move (Y: +24px → 0) + Opacity (0 → 1) → каждому элементу задержка +0.1s

② Плавающие теги (tagFloat)
Что делает:
5 тегов (Brand Safe, Verified, LIVE, AI Compliant, Trend) плавно поднимаются на 8px и опускаются обратно. Каждый с разной скоростью и задержкой — создаёт органичное движение.
В Webflow:
Вставить CSS в Custom Code. В Webflow Interactions нельзя сделать такой loop с разными скоростями.

⑤ Рост шкал при загрузке (barGrow)
Что делает:
Полоски прогресса и score шкалы вырастают от 0 до нужной ширины при загрузке страницы. Ширина у каждой своя (через CSS-переменную --w).
В Webflow:
Interactions → Page Load → для каждой полоски: Width от 0% до нужного % (1.8-2s, ease). Или Custom Code.
